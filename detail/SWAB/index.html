    Instructions

  1. Sign out.
    2. Connect to the DNS
  2. Open the toolbar menu by clicking the time in the bottom right.
  3. In the toolbar menu, click and open the Wi-Fi menu.
  4. Click the settings button to open Wi-Fi settings.
  5. Expand the Network dropdown.
  6. Select Custom name servers.
  7. Set the first input field value to the DNS address.
  8. Disconnect from your network.
  9. Connect to your network.
  10. If a Network not available page appears:

    Click Sign in as an existing user.
