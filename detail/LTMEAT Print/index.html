    Instructions

  1. Find and the longest page of the extension you want to freeze. This is most likely the chrome-extension://extensionid/manifest.json page.
  2. Open your extension's settings page.
  3. Open the print dialog from the context menu or by pressing Ctrl + P.
  4. Increase the number of pages in the print screen by increasing margins.
  5. Reload the page.
  6. Duplicate the tab, and hit the kill switch.
