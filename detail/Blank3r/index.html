Blank3r

(Drag to bookmarks bar)

This is a variation of Point-Blank, the exploit for running scripts on extension pages.

    Instructions

  1. Navigate to chrome://extensions.
  2. Open the details page of a user-installed extension that is installed from the Chrome Web Store.
  3. Copy the 32-character extension ID from the address bar after ?id=.
  4. Click View in Chrome Web Store and spam the Esc key. If the Chrome Web Store loads, try again. Otherwise, if the page is blank, run the bookmarklet.
  5. Paste the extension IDs into the prompt, separated by commas.
If the tab closes, the exploit stops working.
